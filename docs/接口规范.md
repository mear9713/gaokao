# 高考志愿 AI Agent · 接口规范

> v1.0 · 2026-05-29

---

## 基础约定

- **Base URL**: `http://localhost:8000`（开发）
- **请求**: `Content-Type: application/json`，鉴权 `Authorization: Bearer <token>`
- **响应**: `{ code, message, data }`，`code: 0` 成功

---

## 接口清单（9 个）

| # | 方法 | 端点 | 用途 | 鉴权 |
|---|------|------|------|:----:|
| 1 | POST | `/api/auth/login` | 登录 | ❌ |
| 2 | POST | `/api/student/profile` | 提交学生画像 | ✅ |
| 3 | POST | `/api/recommend` | 推荐院校列表 | ✅ |
| 4 | GET  | `/api/school/{id}` | 院校详情（含 5 维评估） | ✅ |
| 5 | POST | `/api/compare` | 横向对比 | ✅ |
| 6 | POST | `/api/agent/chat` | **AI Agent SSE 流式对话** | ✅ |
| 7 | POST | `/api/report/generate` | 生成报告 | ✅ |
| 8 | GET  | `/api/admin/kb` | 知识库列表 | ✅ admin |
| 9 | POST/DELETE | `/api/admin/kb/{id}` | 知识库 CRUD | ✅ admin |

### 6. AI Agent SSE 流式（核心）⭐

```
POST /api/agent/chat
Accept: text/event-stream
```

```json
// 请求
{
  "message": "冲刺西安交大有多大把握？",
  "studentInfo": { /* 同接口 2 */ },
  "history": [{ "role":"user", "content":"...", "timestamp":1716800000000 }],
  "mode": "quick"   // quick(5步) | deep(7步+扩展)
}
```

**SSE 帧格式**：每帧 `data: <JSON>\n\n`

**推送顺序**（严格遵守）：
```
s1.running → s1.done
s2.running → s2.done
...
s5.running → s5.done
(deep 模式追加 s6 / s7)
sources         (一次)
recommendations (一次)
token           (多次)
next_actions    (一次)
done            (最后)
```

**5 个固定步骤**：

| id | icon | label |
|----|------|------|
| s1 | profile  | 正在分析学生画像... |
| s2 | plan     | 正在检索招生计划... |
| s3 | policy   | 正在查询保研政策... |
| s4 | match    | 正在匹配院校专业... |
| s5 | generate | 正在生成升学建议... |

**事件帧示例**：
```
data: {"type":"step","payload":{"id":"s1","status":"running","detail":"识别画像：湖南 568分","cached":false}}

data: {"type":"sources","payload":{"sources":[{"id":"ks_001","type":"录取数据","title":"...","source":"...","excerpt":"..."}]}}

data: {"type":"recommendations","payload":{"recommendations":[...]}}

data: {"type":"token","payload":{"delta":"根据"}}

data: {"type":"next_actions","payload":{"nextActions":["查看详情","加入对比"]}}

data: {"type":"done","payload":{"durationMs":2340}}
```

**注意**：
- 必须用 `sse-starlette`（FastAPI 默认 StreamingResponse 会被 Nginx 缓冲）
- 响应头加 `X-Accel-Buffering: no`
- `s2/s3` 在同场景跟进时 `cached: true`，可秒过节省 token

### 8 & 9. 知识库管理（admin）

```
GET    /api/admin/kb?type=&search=&page=1&pageSize=20
POST   /api/admin/kb       // body: 文档字段（type/title/source/relevance/year/schoolName/excerpt）
DELETE /api/admin/kb/{id}
```

文档字段：`id / type / title / source / relevance / year / schoolName / excerpt`

POST 后端要自动：切片 + Embedding + 入向量库
DELETE 要同时删除对应向量切片

## 枚举速查

| 类型 | 取值 |
|------|------|
| **UserRole** | `student` \| `admin` |
| **RecommendCategory** | `冲刺` \| `稳妥` \| `保底` |
| **AdmissionRisk** | `高` \| `中` \| `低` |
| **RiskPreference** | `冲刺` \| `稳妥` \| `保底` |
| **SchoolPreference** | `985` \| `211` \| `双一流` \| `普通本科` \| `不限` |
| **SubjectType** | `物理` \| `化学` \| `生物` \| `历史` \| `地理` \| `政治` |
| **EducationGoal** | `直接就业` \| `考研` \| `保研` \| `出国留学` \| `考公考编` \| `未定` |
| **KnowledgeSourceType** | `招生数据` \| `录取数据` \| `保研政策` \| `专业信息` \| `培养方案` \| `奖学金政策` \| `转专业政策` \| `院校官网` |
| **PostgradDimensionId** | `opportunity` \| `competition` \| `controllability` \| `extra` \| `destination` |
| **AgentMode** | `quick` \| `deep` |
